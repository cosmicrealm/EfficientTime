# EfficientTime 0.02 Release Notes

## 中文

EfficientTime `0.02` 继续聚焦“把计划变成可执行的一天”。这个版本主要改进了执行面板、状态管理、智能规划体验和发布包。

主要更新：

- 新增右侧 `24 小时状态` 双环钟表盘：内圈表示 `00:00-12:00`，外圈表示 `12:00-24:00`。
- 钟表盘支持点击查看某个时间点的任务或空闲区间，并会同步选中对应时间块。
- 新增更清晰的任务状态归类：已完成、已跳过、已推迟、已删除。
- 删除改为软删除；已删除事项可在 `已删除` 分组里确认后彻底清理。
- 优化状态颜色：已删除为灰色，已跳过使用独立青蓝色，避免语义混淆。
- 优化悬浮窗执行体验：支持更好的透明度控制、折叠/展开、实时秒级倒计时和任务操作。
- 优化智能规划和复盘里的中文输出、AI 服务配置和草稿应用流程。
- 打包脚本现在会把发布版本号写入 `.app` 的 `Info.plist`。
- 更新 README 截图，覆盖新的计划页、智能规划和桌面悬浮窗界面。

下载建议：

- 推荐下载 `EfficientTime-0.02.dmg`。
- 也可以下载 `EfficientTime-0.02-macOS.zip`，解压后运行 `EfficientTime.app`。

注意：当前版本尚未进行 Developer ID 签名和 Apple 公证，首次打开时 macOS 可能会提示安全确认。

## English

EfficientTime `0.02` continues to focus on turning a daily plan into an executable day. This release improves the execution panel, status handling, AI planning flow, and release packaging.

Highlights:

- Added a clickable 24-hour double-ring clock: inner ring for `00:00-12:00`, outer ring for `12:00-24:00`.
- Clicking the clock shows the task or idle window for that time and selects the matching time block.
- Added clearer status grouping: done, skipped, delayed, and deleted.
- Deletion is now a soft-delete flow; deleted items can be permanently cleared after confirmation.
- Improved status colors: deleted is gray, skipped uses a distinct cyan-blue color.
- Improved floating execution panel behavior: opacity control, compact/expanded interaction, second-level countdown, and direct task actions.
- Improved Chinese AI review/planning output, AI provider settings, and draft application flow.
- Packaging scripts now write the release version into the `.app` `Info.plist`.
- Refreshed README screenshots for the updated plan, AI planning, and floating panel UI.

Download:

- Recommended asset: `EfficientTime-0.02.dmg`.
- Alternative asset: `EfficientTime-0.02-macOS.zip`.

Note: this build is not Developer ID signed or Apple notarized yet, so macOS may show a security confirmation on first launch.
