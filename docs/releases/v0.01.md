# EfficientTime 0.01 Release Notes

## 中文

EfficientTime `0.01` 是早期本地预览版，核心目标是把每天的计划变成可以执行、可以提醒、可以在桌面持续查看的时间表。

主要功能：

- 本地单机运行，不需要账户和云同步。
- 支持按日期规划任务，并在主窗口查看当天时间表。
- 支持桌面悬浮窗显示当前任务、倒计时和附近任务。
- 支持任务开始和结束时的系统通知提醒。
- 支持快速添加任务、完成、跳过和推迟。
- 支持 DeepSeek 和火山方舟作为智能规划服务，把原始计划文本生成可编辑日程草稿。

下载建议：

- 推荐下载 `EfficientTime-0.01.dmg`。
- 也可以下载 `EfficientTime-0.01-macOS.zip`，解压后运行 `EfficientTime.app`。

注意：当前版本尚未进行 Developer ID 签名和 Apple 公证，首次打开时 macOS 可能会提示安全确认。

## English

EfficientTime `0.01` is an early local preview focused on turning a daily plan into an executable schedule that stays visible and sends real-time reminders.

Highlights:

- Local-first macOS app with no account or cloud sync requirement.
- Date-based task planning and daily schedule view.
- Always-on floating panel for the current task, countdown, and nearby tasks.
- macOS notifications when tasks start and end.
- Quick task entry, done, skip, and delay actions.
- AI planning draft generation with DeepSeek and Volcengine Ark.

Download:

- Recommended asset: `EfficientTime-0.01.dmg`.
- Alternative asset: `EfficientTime-0.01-macOS.zip`.

Note: this build is not Developer ID signed or Apple notarized yet, so macOS may show a security confirmation on first launch.
