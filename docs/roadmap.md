# EfficientTime Roadmap

## v0.1 工程骨架

- Swift Package 工程。
- Core 领域模型。
- 本地排程引擎。
- AI 规划协议和 DeepSeek 预留实现。
- 主窗口、菜单栏、悬浮窗骨架。
- JSON 本地存储。
- 核心单元测试。

## v0.2 可用计划编辑

- 添加/编辑/删除任务。
- 手动拖拽或表单调整时间块。
- 开始今日计划。
- 系统通知完整接入。

当前实现采用表单调整时间块，尚未做拖拽。

## v0.3 AI Plan Builder

- DeepSeek API 接入。
- 自然语言拆分任务。
- 规划澄清问题。
- AI 草稿确认界面。
- 隐私级别和脱敏预览。

## v0.4 执行复盘

- 今日完成率。
- 延迟和跳过分析。
- 未完成任务继承。
- 脱敏 AI 日终总结。

当前实现包含本地复盘摘要；AI 日终总结接口已在 Core 层预留。

## 暂缓需求池

- ChatGPT 手动桥接模式：在没有 OpenAI API Key 时，由 App 生成可复制到 ChatGPT 的规划 Prompt，再手动粘回 JSON 结果并解析为草稿。当前暂缓，不接入产品逻辑；不做网页登录自动化，不尝试复用 ChatGPT/Codex 登录态或额度。
